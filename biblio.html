Transformers for Medical Image Segmentation -------------------------------------------------------- [repository]
In this project I implemented several pipelines for medical image segmentation pipelines using Transformers ( MedT, Vit, TransUNet) and then compared them and validate these pipelines in the context of abdominal image segmentation using the Chaos dataset Once the best architecture has been identified, a derived architecture with multiple heads (one for each organ) will be implemented in order to exploit both generic and representations of abdominal images specific to each organ. to see the github repository [pytorch] [UNet] [TransUNet] [MedT]
Object Localization with Tensorflow : emojis detection and localization----------------------------------- [repository]
Used Tensorflow to create a model which will be trained to classify and localized emojis in the input images. Localization means the position off the emojis in the images, so the model will have one input and two outputs.
NLP application : Text classification of BBC news dataset----------------------------------------------- [repository]
We will be using the "BBC-news" dataset available in Kaggle, firstly, we will pre-process the dataset ( Cleaning, instance selection, normalization, feature extraction, and selection) and then build three types of models to classify sentences into 5 categories of vocabulary ( Tech, business, sport, entertainment, and politics ) secondly, we will compare models performance and visualize word embedding in 2D using PCA and in 3D using T-SNE.
